SQL SERVER DATABASE ENGINE IS USED TO CREATE DATABASES AND TO QUERY THEM.
SSIS IS USED FOR INTEGRATIONS OR BUILD DATA PIPELINES. THAT's WHY NO NEED TO CREATE A LOGIN FOR SSIS SERVICE IN SQL SERVER.
SSAS IS USED FOR CREATING DATA MODELS AND CUBE FORMATIONS AND IT ALSO UTILZES DATABASE LIKE MODEL TO STORE DATA. ITS 
  SOMEHOW ALIKE DATABASE ENGINE AND SSAS SERVISE HAS ITS OWN SERVER TO WORK ON. THAT's WHY NEED TO CREATE A LOGIN FOR 
  SSAS SERVISE IN SQL SERVER LOGINS.
SSRS IS USED TO BUILD REPORTS OR VISUALIZED CONTENTS. THAT's WHY NO NEED TO CREATE A LOGIN FOR SSIS SERVICE IN SQL SERVER.
